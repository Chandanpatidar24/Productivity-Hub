# 🧠 Productivity Hub

A personal productivity web app to manage your **tasks**, write **notes**, and stay focused.  
Built with **React + Vite + TailwindCSS + shadcn/ui** for speed and clean design.

---

## 🌟 Features (Progress Overview)

### 📋 Tasks Module
- ✅ Add new tasks (title, description, time, day)
- ✅ Edit and delete tasks
- ✅ Mark tasks as done / pending
- ✅ Filter by Done / Pending
- ✅ Modal form for Add/Edit
- ✅ Fully responsive layout
- ⬜ Save tasks to Firebase
- ⬜ Dark mode
- ⬜ More features.....

### 🗒️ Notes Module
- ✅ Add notes (title + content)
- ✅ Edit and delete notes
- ✅ Modal-based note form
- ✅ Styled note cards (UX/UI)
- ✅ Responsive layout
- ⬜ Deadline/Reminder support
- ⬜ Notes search/filter
- ⬜ Sync notes to Firebase

### ⏲️ Timer (Upcoming)
- ✅ Start / Pause timer functionality
- ✅ Reset button to reset the current timer
- ✅ Pomodoro Mode (25 min) Short Break (5 min) & Long Break (10 min)
- ✅ Countdown timer format (MM:SS)
- ✅ Clean React Hooks structure (uses useState, useEffect, useRef)
- ⬜ Sound alert on timer end (Coming up)
- ⬜ Auto switch between Pomodoro and Breaks (Coming up)
